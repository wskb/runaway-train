# Runaway Train

An interactive multimedia experience

Written in Applesoft BASIC for the Apple ][e

## Context

My employer, Onware, had a contest to write an Applesoft Basic program.

Applesoft Basic has limitations - only the first two letters of variable names are
significant; there's no if/else, you have to number lines, etc. So I wrote a
typescript program to write the basic for me.
